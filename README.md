# 2022 - djangogram Project 

## Description

**Project Name :** djangogram

instagram의 기본 기능 클론 프로젝트입니다.

## Function
 - 회원가입, 로그인, 로그아웃
 - 좋아요, 좋아요 취소
 - 게시물 CRUD
 - 댓글 CRUD

## Environment

**Web Framework :** Django

**Database sys :** SQLite
